r1432 | phanx | 2011-03-14 05:26:20 +0000 (Mon, 14 Mar 2011) | 6 lines
Changed paths:
   A /tags/1.40000.1432-beta (from /trunk:1431)

- Range checking has been revamped. There is now a single "Out of Range" status that works for all classes and specs. The old "More than X yards away" statuses have been removed. If you were one of the few people who wanted to check more than one range at a time, use the GridStatusMultiRange plugin.
- When typing in the Add Buff and Add Debuff fields, an auto-completion dropdown will now appear with the names of spells that match what you've typed.
- The drag tab will now open the options window when right-clicked.
- Fixed the Layout > Border Color option.
- For Plugin Authors: The SendStatusGained API now accepts an additional parameter for specifying texture coordinates. This parameter should be a table with numeric values for "left", "right", "top", and "bottom", similar to the structure of the color table parameter.
- For Translators: Localization is now handled through the WowAce localization system. To add or update translations, please visit the project page on wowace.com.
